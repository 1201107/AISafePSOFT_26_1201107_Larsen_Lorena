# Glossary

| Concept                                                 | Description                                                                                                                                 |
|---------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| **Airport/Aeroporto**                                   | A location where aircraft operate, identified by a unique IATA code, providing runways, facilities, certifications, and operational status. |
| **Runway/Pista**                                        | A physical strip within an airport used for takeoff and landing, with defined dimensions, orientation, and operational status.              |
| **Airport Location/ Localização Aeroporto**             | Geographic and regional data describing an airport.                                                                                         |
| **Facilities/Edifícios de Serviços**                    | Structured representation of airport infrastructure.                                                                                        |
| **Contact/Contactos**                                   | A contact info for an airport.                                                                                                              |
| **Certification/Certificados**                          | A formal authorization, compliance record or norm associated with an airport, including category and expiration date.                       |
| **Airport Status/Status do Aeroporto**                  | Operational state of an airport.                                                                                                            |
| **Runway Status/Status da Pista**                       | Operational state of a runway.                                                                                                              |
| **Contact Type/Tipo de Contacto**                       | Type of communication method.                                                                                                               |
| **Flight/Voo**                                          | A scheduled execution of a route using a specific aircraft.                                                                                 |
| **Flight Status/Status de Voo**                         | State of a flight.                                                                                                                          |
| **Route/Rota**                                          | A defined connection between airports, including operational status, type, and estimated flight time.                                       |
| **Route Requirements/Requirimentos da Rota**            | Constraints that a route needs to be available.                                                                                             |
| **Route Status/Status da Rota**                         | Operational status of a Route.                                                                                                              |
| **Route Type/Tipo de Rota**                             | Route type.                                                                                                                                 |
| **Aircraft Model/Modelo de Avioneta**                   | A model of aircraft describing manufacturer-level characteristics and general properties.                                                   |
| **Aircraft Specifications/Especificações de Avioneta**  | Technical capabilities of an aircraft model.                                                                                                |
| **Aircraft/Avioneta**                                   | A physical aircraft instance.                                                                                                               |
| **Installed Component/Componente Instalado**            | A physical component or part installed on an aircraft.                                                                                      |
| **Aircraft Availability/Disponibilidade da Avioneta**   | Operational status of an aircraft.                                                                                                          |
| **Maintenance Record/Registo de Maintenance**           | A record of a maintenance activity performed on an aircraft.                                                                                |
| **Maintenance Template/Template de Maintenance**        | A predefined procedure or checklist that defines how a maintenance operation should be performed.                                           |
| **Maintenance Operation/Operação de Manutenção**        | Type of maintenance activity.                                                                                                               |
| **Maintenance Status/Status de Manutenção**             | Progress state of a maintenance activity.                                                                                                   |
| **Maintenance Attribute/Atributo que sofre Manutenção** | The area of the aircraft targeted by maintenance.                                                                                           |
| **Technician/Técnico**                                  | A maintenance worker responsible for executing maintenance tasks.                                                                           |
| **Supervisor/Supervisor**                               | A person responsible for overseeing maintenance operations and ensuring proper execution.                                                   |
| **Used Part/Parte Usada na Manutenção**                  | A component or part consumed or replaced during a maintenance activity, including quantity and cost.                                        |
