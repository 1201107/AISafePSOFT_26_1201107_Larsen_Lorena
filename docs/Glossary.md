# Glossary

| Concept                     | Description                                                                                                                                 |
|-----------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| **Airport**                 | A location where aircraft operate, identified by a unique IATA code, providing runways, facilities, certifications, and operational status. |
| **Runway**                  | A physical strip within an airport used for takeoff and landing, with defined dimensions, orientation, and operational status.              |
| **Airport Location**        | Geographic and regional data describing an airport.                                                                                         |
| **Facilities**              | Structured representation of airport infrastructure.                                                                                        |
| **Contact**                 | A contact info for an airport.                                                                                                              |
| **Certification**           | A formal authorization, compliance record or norm associated with an airport, including category and expiration date.                       |
| **Airport Status**          | Operational state of an airport.                                                                                                            |
| **Runway Status**           | Operational state of a runway.                                                                                                              |
| **Contact Type**            | Type of communication method.                                                                                                               |
| **Flight**                  | A scheduled execution of a route using a specific aircraft.                                                                                 |
| **Flight Status**           | State of a flight.                                                                                                                          |
| **Route**                   | A defined connection between airports, including operational status, type, and estimated flight time.                                       |
| **Route Requirements**      | Constraints that a route needs to be available.                                                                                             |
| **Route Status**            | Operational status of a Route.                                                                                                              |
| **Route Type**              | Route type.                                                                                                                                 |
| **Aircraft Model**          | A model of aircraft describing manufacturer-level characteristics and general properties.                                                   |
| **Aircraft Specifications** | Technical capabilities of an aircraft model.                                                                                                |
| **Aircraft**                | A physical aircraft instance.                                                                                                               |
| **Installed Component**     | A physical component or part installed on an aircraft.                                                                                      |
| **Aircraft Availability**   | Operational status of an aircraft.                                                                                                          |
| **Maintenance Record**      | A record of a maintenance activity performed on an aircraft.                                                                                |
| **Maintenance Template**    | A predefined procedure or checklist that defines how a maintenance operation should be performed.                                           |
| **Maintenance Operation**   | Type of maintenance activity.                                                                                                               |
| **Maintenance Status**      | Progress state of a maintenance activity.                                                                                                   |
| **Maintenance Attribute**   | The area of the aircraft targeted by maintenance.                                                                                           |
| **Technician**              | A maintenance worker responsible for executing maintenance tasks.                                                                           |
| **Supervisor**              | A person responsible for overseeing maintenance operations and ensuring proper execution.                                                   |
| **Used Part**               | A component or part consumed or replaced during a maintenance activity, including quantity and cost.                                        |
