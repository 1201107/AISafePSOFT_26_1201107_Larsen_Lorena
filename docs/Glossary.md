# Glossary

| Concept                                                  | Description                                                                                                                                                       |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Airport / Aeroporto**                                  | An operational aviation facility identified by a unique IATA code, containing runways, facilities, certifications, contacts, and geographic information.          |
| **Runway / Pista**                                       | A landing and takeoff strip within an airport, identified by a runway ID, with dimensions, orientation, and operational status.                                   |
| **Airport Location / Localização do Aeroporto**          | A value object containing the geographic coordinates, timezone, city, country, and regional information of an airport.                                            |
| **Facilities / Infraestruturas**                         | A structured representation of airport infrastructure, including terminals, gates, and available services.                                                        |
| **Contact / Contacto**                                   | Contact information associated with an airport, such as phone numbers, email addresses, fax, or social media accounts.                                            |
| **Certification / Certificação**                         | A compliance or authorization record associated with an airport, including validity period and category.                                                          |
| **Airport Status / Estado do Aeroporto**                 | The operational condition of an airport, such as operational, closed, or under maintenance.                                                                       |
| **Runway Status / Estado da Pista**                      | The operational condition of a runway.                                                                                                                            |
| **Contact Type / Tipo de Contacto**                      | The communication channel used for a contact entry.                                                                                                               |
| **Flight / Voo**                                         | A scheduled operation connecting a route with a specific aircraft, including scheduling and operational status information.                                       |
| **Flight Status / Estado do Voo**                        | The current lifecycle state of a flight, such as scheduled, delayed, departed, canceled, or arrived.                                                              |
| **Route / Rota**                                         | A defined air connection between two airports, including route requirements, operational status, type, and historical usage information.                          |
| **Route History / Histórico da Rota**                    | Historical operational information for a route, including start date, finish date, and usage metrics.                                                             |
| **Route Requirements / Requisitos da Rota**              | Operational constraints required for a route, such as aircraft range and seating capacity.                                                                        |
| **Route Status / Estado da Rota**                        | The operational availability of a route.                                                                                                                          |
| **Route Type / Tipo de Rota**                            | The classification of a route, such as direct or scaled.                                                                                                          |
| **Aircraft Model / Modelo de Aeronave**                  | A catalog-level representation of an aircraft model containing manufacturer information, specifications, and model images.                                        |
| **Aircraft Specifications / Especificações da Aeronave** | Technical capabilities of an aircraft model, including fuel capacity, range, and cruising speed.                                                                  |
| **Aircraft / Aeronave**                                  | A physical aircraft instance identified by a registration number, containing operational metrics, certifications, seating configuration, and installed components. |
| **Seating Pack / Configuração de Assentos**              | A seating configuration defining seating capacity and orientation inside an aircraft.                                                                             |
| **Installed Component / Componente Instalado**           | A physical component installed on an aircraft, identified by serial and part numbers with its operational status.                                                 |
| **Aircraft Certification / Certificação da Aeronave**    | A compliance or authorization record associated specifically with an aircraft.                                                                                    |
| **Aircraft Availability / Disponibilidade da Aeronave**  | The operational condition of an aircraft, such as available, in flight, inactive, or under maintenance.                                                           |
| **Maintenance Record / Registo de Manutenção**           | A maintenance operation record associated with an aircraft, including technicians, duration, status, used parts, and completion details.                          |
| **Maintenance Template / Modelo de Manutenção**          | A predefined maintenance procedure defining expected duration, checklist, maintenance attribute, and operation type.                                              |
| **Done List / Lista de Tarefas Executadas**              | A value object storing maintenance completion notes and executed checklist tasks.                                                                                 |
| **Maintenance Type / Tipo de Manutenção**                | The classification of a maintenance operation, such as inspection, overhaul, modification, or scheduled maintenance.                                              |
| **Maintenance Status / Estado da Manutenção**            | The progress state of a maintenance activity, such as ongoing, completed, or inline.                                                                              |
| **Maintenance Attribute / Área de Manutenção**           | The aircraft subsystem or area targeted during maintenance, such as engine, avionics, or airframe.                                                                |
| **Used Part / Peça Utilizada**                           | A part consumed or replaced during maintenance.                                                                       |
| **Collaborator / Colaborador**                           | A system user involved in operational activities such as maintenance execution, supervision or using the system.                                                  |
| **Technician / Técnico**                                 | A collaborator responsible for performing maintenance tasks on aircraft.                                                                                          |
| **Supervisor / Supervisor**                              | A collaborator responsible for overseeing and validating maintenance operations.                                                                                  |
