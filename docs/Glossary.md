# Aviation System Glossary

| Concept                  | Description                                                |
|--------------------------|------------------------------------------------------------|
| **Aircraft Model**       | Defines specifications shared by aircraft.                 |
| **Aircraft**             | Physical airplane with unique registration.                |
| **Airport**              | Location where aircraft operate.                           |
| **Route**                | Path between two airports with operational constraints.    |
| **Scheduled Flight**     | A specific execution of a route at a scheduled time.       |
| **Maintenance Template** | Predefined maintenance procedure used for aircraft checks. |
| **Maintenance Record**   | Execution of maintenance on a specific aircraft.           |
| **Checklist**            | List of tasks to complete during maintenance.              |
| **Certification**        | Approval of a specific aircraft model at an airport.       |
| **User Role**            | Defines permissions and access levels for system users.    |
